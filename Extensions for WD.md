# Extensions for WD
Extenstions are add-ons that help to improve the workflow and productivity of developers while working on web development projects. Here are some popular extensions for web development:


## Eslint:
Eslint is a static code analysis tool for identifying problematic patterns in code. It helps developers write cleaner and more consistent code by enforcing coding standards and best practices.

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint

## Prettier:
Prettier is an opinionated code formatter that automatically formats code to ensure a consistent style across a project. It supports many programming languages and integrates with various editors.

VS Marketplace Links:
https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode
https://marketplace.visualstudio.com/items?itemName=rvest.vs-code-prettier-eslint

## Code Spell Checker:
Code Spell Checker is a simple spell checker for source code. It helps developers catch common spelling errors in their code comments, strings, and identifiers.

https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker


## colorize
A vscode extension to help visualize css colors in files.

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=kamikillerto.vscode-colorize

## Auto Import
Automatically finds, parses and provides code actions and code completion for all available imports. Works with Typescript and TSX

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=steoates.autoimport

## ES7+ React/Redux/React-Native snippets
Extensions for React, React-Native and Redux in JS/TS with ES7+ syntax. Provides various snippets for faster development.

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets

## Live Server:
Live Server is an extension that launches a local development server with live reload feature for static and dynamic pages. It allows developers to see changes in real-time as they edit their code.

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

## Auto Rename Tag
an extension that automatically renames paired HTML/XML tags. When you rename one tag, the corresponding closing tag is renamed automatically.

VS Marketplace Link: https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag

## Postman:
Postman is a popular API development and testing tool that allows developers to create, test, and document APIs. It provides a user-friendly interface for sending requests and analyzing responses.

https://marketplace.visualstudio.com/items?itemName=Postman.postman-for-vscode
